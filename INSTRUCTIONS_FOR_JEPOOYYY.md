# Instructions for jepooyyy to Clone and Push to StainScanning

These are the steps to clone the StainScan code from Kr1stik's fork and push it to your repository.

## Quick Setup

```bash
# 1. Clone Kr1stik's repository
git clone https://github.com/Kr1stik/Stainscan.git StainScanning
cd StainScanning

# 2. Add your repository as a remote
git remote add jepooyyy https://github.com/jepooyyy/StainScanning.git

# 3. Push to your repository
git push -u jepooyyy main

# (Optional) Push all branches and tags
git push -u jepooyyy --all
git push -u jepooyyy --tags
```

## What This Does

- Clones the complete StainScan project from Kr1stik's fork
- Sets up your repository as a remote
- Pushes all commits to your repository at `https://github.com/jepooyyy/StainScanning`

- **Source (Kr1stik's fork):** https://github.com/Kr1stik/Stainscan

You'll have complete ownership of the code in your repository. Both repositories will contain the same codebase, and you can make changes independently from Kr1stik's fork.
